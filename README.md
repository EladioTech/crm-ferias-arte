# CRM Ferias de Arte - Seguimiento

Este es un CRM sencillo y funcional para gestionar la participación en ferias de arte. Permite añadir ferias/galerías, marcar el estado con semáforos, añadir comentarios y manejar los datos desde la interfaz.

## Funcionalidades principales

- **Buscar**: Filtra por nombre de galería o feria.
- **Añadir fila**: Crea una nueva entrada.
- **Resetear semáforos**: Reinicia todos los semáforos a “Sin marcar”.
- **Importar JSON / Exportar JSON**: Permite guardar y recuperar la información.
- **Navegación por años**: ⬅️ Año anterior / Año siguiente ➡️.
- **Semáforos de estado**: Rojo 🔴, Amarillo 🟡, Verde 🟢.
- **Acciones por fila**: Subir / bajar fila y cambiar semáforo desde la interfaz.

## Uso

1. Abrir `index.html` en cualquier navegador moderno.
2. Utilizar la barra de búsqueda para filtrar las ferias.
3. Añadir nuevas filas o modificar existentes.
4. Guardar los datos con **Exportar JSON** o cargar con **Importar JSON**.

## Notas

- Los cambios solo se guardan en la sesión actual a menos que uses **Exportar JSON**.
- Evita subir los archivos con datos reales al repositorio, por eso se incluye `.gitignore`.
